docker build -t simplebank:latest
to reduce image size we use multistage build command 

to run the simplebank container
docker run --name simplebank -p 8080:8080 simplebank:latest
docker run --name simplebank -p 8080:8080 -e GIN_MODE=release simplebank:latest

CONNECT TWO INDEPENDENT CONTAINER 

1. BAD PRACTICE-> because ip will change after every restart 
docker run --name simplebank -p 8080:8080 -e GIN_MODE=release -e DB_SOURCE="postgresql://root:secret@172.17.0.3:5432/simplebank?sslmode=disable" simplebank:latest 

2. BUILD DOCKER NETWORK :- a network where all other container will interact 
docker network create bank-network  
docker network connect bank-network postgres12
docker run --name simplebank --network bank-network -p 8080:8080 -e GIN_MODE=release -e DB_SOURCE="postgresql://root:secret@postgres12:5432/simplebank?sslmode=disable" simplebank:latest
